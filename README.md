# 🤖 AI Code Debugger App (Streamlit + Gemini)

An AI-powered web application that detects errors from code screenshots and provides intelligent debugging suggestions using **Google Gemini API**.

Built with **Streamlit, Python, and Gemini AI**, this project helps developers quickly understand bugs, get hints, or receive full corrected solutions.

---

## 🚀 Features

- 📤 Upload code error screenshots (PNG, JPG, JPEG)
- 🧠 AI-powered error detection using Gemini API
- 💡 Two modes:
  - **Hints Mode** (guided debugging)
  - **Solution Mode** (full corrected code)
- ⏳ Loading spinner during AI processing
- ⚠️ Input validation with error alerts
- 🔐 Secure API key management using `.env`
- 📝 Clean and readable AI response display
